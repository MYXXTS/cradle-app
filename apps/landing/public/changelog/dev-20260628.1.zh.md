## ✨ 新功能

- Tear-off 窗口支持任意 Surface — Chat、Workspace、Diffs、Kanban、Plugin 视图均可分离为独立 Electron 窗口

## 💎 改进

- Diff Review local-working-tree 保持 open 状态并显示 Live 标记，更准确地反映未提交变更的实时快照性质
- Diff Review agent fixes 现在绑定到创建时的 revision，避免过期 planning 泄漏到后续版本
- Composer 跨 tab 切换保持草稿
- Codex diagnostics 显示 reset-credit 数量并去重 rate-limit 卡片
- Settings 不再出现在 surface tabs 中
- Fuzzy model lookup 改用 token-aligned 前缀匹配，结果更准确

## 🐞 修复

- 修复非手动 session 未始终保持 dim 状态的问题
- 修复更新暂存目录未正确隔离的问题
- 修复 Windows 安装包 server runtime 打包问题
- 移除 tray 标题的 health alert badge
